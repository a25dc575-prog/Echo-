🌟 プラットフォームについて
Echo は、音楽の好みでつながり、ライブやコンサートをきっかけに交流を深め、リアルタイムでチャットできる音楽ソーシャルプラットフォームです。モダンなダークモードデザインを採用し、日本語と英語の切り替えもスムーズに行えます。
主な機能

🎨 ブランドカラー:Cyber Teal(#17D1C1)——テック感のあるブルーグリーン
🌓 テーマ切り替え:ライトモードとダークモードをシームレスに切り替え
🌍 多言語対応:日本語と英語をワンタップで切り替え可能
📱 レスポンシブデザイン:PC、タブレット、スマホすべてに最適化
🎵 音楽 DNA:あなたの音楽の好みをレーダーチャートで視覚化
👥 ソーシャルマッチング:音楽の趣味が合う人と出会える
🎤 コンサート機能:同じライブに参加する人を見つけられる
💬 チャットシステム:ダークモード対応、音楽カードも共有できる
